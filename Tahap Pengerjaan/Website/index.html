Deskripsi Proyek

Proyek Kelompok 4 Praktikum Kecerdasan Buatan ini bertujuan untuk membangun sistem Analisis Sentimen Ulasan Pelanggan E-Commerce menggunakan metode Machine Learning dengan implementasi berbasis web application. Sistem dikembangkan menggunakan dataset ulasan pelanggan dari platform e-commerce seperti Tokopedia dan Shopee untuk mengidentifikasi opini atau emosi pengguna terhadap suatu produk. Aplikasi ini menerima input berupa teks ulasan pelanggan, kemudian memproses dan mengklasifikasikan sentimen ke dalam tiga kategori yaitu Positif, Negatif, dan Netral. Selain klasifikasi sentimen, sistem juga menampilkan tingkat kepercayaan (confidence score) hasil prediksi, analisis kata-kata yang sering muncul pada ulasan positif maupun negatif, serta visualisasi data berupa pie chart dan wordcloud. Proses pengembangan sistem dilakukan melalui beberapa tahapan, yaitu pengambilan dataset, pembersihan data, pelabelan otomatis berdasarkan rating produk, text preprocessing seperti tokenization dan stemming, ekstraksi fitur menggunakan TF-IDF, pelatihan model Machine Learning seperti Naive Bayes, Logistic Regression, atau Support Vector Machine, serta evaluasi performa model menggunakan confusion matrix, accuracy, precision, recall, dan F1-score. Hasil akhir dari proyek ini berupa aplikasi web interaktif yang memungkinkan pengguna memasukkan ulasan produk dan memperoleh hasil analisis sentimen secara langsung sehingga dapat membantu memahami opini pelanggan terhadap suatu produk atau layanan.
